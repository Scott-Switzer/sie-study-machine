# SIE Study Machine

A self-contained web app to drill for the FINRA **Securities Industry Essentials (SIE)** exam.

- **2,100+ practice questions** aggregated from free sources (YouTube walkthroughs, Udemy practice test, curated banks)
- **Practice mode** — drill by section or mixed, with explanations after each answer
- **Timed Mock Exam** — 75 scored + 5 pretest questions, FINRA blueprint-weighted (12/33/23/7), 105 min
- **Flashcards** — active recall
- **Literature** — offline SIE study guides & the official FINRA content outline
- **Progress tracking** — accuracy by section, weak-area resurfacing (saved in your browser)

## Use it
Just open the site — everything runs in your browser, no login, works offline once loaded, works on phone.

## Run locally
Open `index.html` in a browser, or serve it:
```bash
python3 -m http.server 8799
# then visit http://localhost:8799
```
